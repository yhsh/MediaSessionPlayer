# MediaSessionPlayer

#### Description
MediaSession多媒体框架demo源码，可将源码中\app\src\main\assets这个资源文件夹中的mp3歌曲赋值到你的手机,然后运行项目跑起来，APP会自动扫描手机里面的所有歌曲，扫描成功后会展示歌曲列表，默认点击播放播放第一首歌曲，也可以点击歌曲列表中的任意歌曲开始播放
